Расписание

Дополнительные rclone-опции
Поддерживаются маски `rclone`: `*` любая часть имени, `?` один символ, `**` любой путь. Примеры: `*frigate*`, `report_2026-??-??.*`, `**/cache/**`.
Можно добавить каталог или файл внутри исходного каталога. Для каталога будет исключено всё содержимое.
Можно писать несколько токенов в одной строке. Кавычки поддерживаются, backend разбивает строку как shell argv.

Очистка в облаке
Те же маски `rclone`: `*`, `?`, `**`. Примеры: `keep/**`, `*.keep`, `report_2026-??-??.*`.

Команда

Работает для command-задач, у которых первая строка команды — `rclone`. Backend сам добавит `--log-file` к step-логу.
