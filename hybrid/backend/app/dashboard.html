Rclone лог
Когда выключено, прогресс берётся из stdout/stderr процесса без `--log-file`. Когда включено, для новых запусков создаются отдельные логи по шагам в `hybrid/backend/data/rclone-logs`.
Watcher
Watcher работает внутри backend и следит только за теми backup-задачами, у которых включено наблюдение за исходным каталогом.
Gotify
Общие параметры Gotify для всех задач. Локальные настройки уведомлений задаются внутри карточки задания.
